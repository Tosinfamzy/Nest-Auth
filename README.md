## Description

Auth Solutions examples with [NestJS](https://docs.nestjs.com)

## Project setup

```bash
$ npm install
$ docker-compose up
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Feats

- Sign in & Sign out via
  - Username/password
  - Google Auth
- JWT authentication
- Role-based access control
